## 0. 整体架构结论

### 0.1 现状诊断

经过对 268 个 PHP 文件、57 个 Model、70+ Service、15 个 RPA 脚本的完整审计，核心结论：

| 维度 | 现状评分 | 关键卡点 |
|------|---------|---------|
| 多租户架构 | 8/10 | 22 个控制器缺 workspace 鉴权(P0 已修复) |
| AI 调用链路 | 6/10 | 单一 `api_url + model_id` 模式，无 provider 抽象层 |
| RPA 自动化 | 7/10 | 15 个成熟脚本，但发布注册闭环断裂 |
| 代码质量 | 6.5/10 | 上帝方法 + 重复代码多，但命名规范好 |
| 性能 | 5.5/10 | 缺关键索引，N+1 趋势查询 |

### 0.2 核心设计原则

1. **现有资产零推翻**：所有方案基于现有 Laravel 12 + PostgreSQL + Redis + Horizon 架构
2. **复用优先**：新增代码目标复用 ≥70% 现有 Service/Model
3. **PHP 原生优先**：优先用 PHP Service + Eloquent + Horizon 队列
4. **租户隔离贯穿**：所有新增模块必须携带 workspace_id
5. **安全体系统一**：复用 AES-256-CBC 加密、参数化查询、现有文件校验
6. **MCP 渐进引入**：从零开始规划，不假设任何 MCP 已安装

### 1.1 AgentDispatcherService — 顶层调度器（新增）

**文件**：`app/Services/Agent/AgentDispatcherService.php`

```php
class AgentDispatcherService
{
    // 状态机定义
    const STATE_IDLE       = 'idle';        // 空闲，等待触发
    const STATE_SCOUTING   = 'scouting';    // 侦察中
    const STATE_PLANNING   = 'planning';    // 策略规划中
    const STATE_WRITING    = 'writing';     // 内容生产中
    const STATE_DEPLOYING  = 'deploying';   // 分发执行中
    const STATE_REVIEWING  = 'reviewing';   // 复盘分析中
    const STATE_COMPLETED  = 'completed';   // 全流程完成
    const STATE_FAILED     = 'failed';      // 失败（含重试耗尽）

    // 状态转换表
    const TRANSITIONS = [
        self::STATE_IDLE      => [self::STATE_SCOUTING, self::STATE_PLANNING],
        self::STATE_SCOUTING  => [self::STATE_PLANNING, self::STATE_FAILED],
        self::STATE_PLANNING  => [self::STATE_WRITING, self::STATE_FAILED],
        self::STATE_WRITING   => [self::STATE_DEPLOYING, self::STATE_WRITING], // WRITING可自循环(GEO低分重写)
        self::STATE_DEPLOYING => [self::STATE_REVIEWING, self::STATE_DEPLOYING], // DEPLOYING可自循环(多轮分发)
        self::STATE_REVIEWING => [self::STATE_COMPLETED, self::STATE_PLANNING], // 复盘后可重新规划
    ];

    // 核心方法
    public function dispatch(AgentTask $task, string $targetState): AgentExecution;
    public function transition(AgentExecution $execution, string $nextState): AgentExecution;
    public function getAvailableTransitions(AgentExecution $execution): array;
}
```

**队列协同机制**：
- 侦察完成后投递策略任务 → `ScoutCompletionEvent` → `DispatchStrategyJob`
- 策略完成后投递内容任务 → `PlanReadyEvent` → `DispatchContentJob`
- 内容完成后投递分发任务 → `ContentReadyEvent` → `DispatchDeployJob`
- 分发完成后投递复盘任务 → `DeployCompleteEvent` → `DispatchReviewJob`
- 所有事件走 Laravel Event 系统，用 `ShouldDispatchAfterCommit` 保证事务一致性

### 2.2 抽象基类 + 各厂商 Adapter 设计

```php
namespace App\Services\AI\Adapters;

// 抽象基类
abstract class BaseLlmAdapter
{
    protected AiModelProvider $provider; // 关联 ai_model_providers 表
    protected string $apiKey;            // 解密后的 API Key

    // 每个厂商必须实现
    abstract public function chat(array $messages, array $options): array;
    abstract public function chatStream(array $messages, array $options): Generator;
    abstract public function embedding(string $text, string $model): array;
    abstract public function testConnection(): bool;

    // 参数标准化：把内部统一参数转为厂商特定参数
    abstract protected function normalizeOptions(array $options): array;

    // 响应标准化：把厂商特定响应转为统一格式
    abstract protected function normalizeResponse(array $raw): array;

    // 通用方法
    protected function getDecryptedApiKey(): string;
    protected function httpClient(): PendingRequest;
}

// 火山方舟适配器
class VolcengineArkAdapter extends BaseLlmAdapter
{
    // 特点：豆包模型 + OpenAI兼容 + context caching
    protected function normalizeOptions(array $options): array
    {
        // max_tokens → max_tokens (同名，无需转换)
        // temperature → temperature
        // 唯一特殊：stream_options 需包含 include_usage
        return $options;
    }
}

// 通义千问适配器
class QwenDashScopeAdapter extends BaseLlmAdapter
{
    // 特点：OpenAI兼容 + 阿里百炼平台
    // 差异：需设置 X-DashScope-SSE: enable 启用流式
    protected function normalizeOptions(array $options): array
    {
        // max_tokens → max_tokens (同名)
        return $options;
    }

    protected function httpClient(): PendingRequest
    {
        return parent::httpClient()
            ->withHeader('X-DashScope-SSE', 'enable');
    }
}
```

### 2.4 智能故障切换 (smart_failover)

```
优先级链：（按 ai_model_providers.failover_priority 排序 + 实时健康状态）

1. 尝试主模型 → 成功？→ 返回结果
    ↓ 失败
2. 降级模型1（同 provider 不同 model_id）
    ↓ 失败
3. 降级模型2（不同 provider，如 DeepSeek → 硅基流动 → 火山方舟）
    ↓ 失败
4. 最终兜底模型（config('geoflow.fallback_llm_model')）

重试策略：
- 同一模型：最多 2 次（排除 4xx 认证/参数错误）
- 指数退避：1s → 2s → 4s
- 熔断：5 分钟内连续失败 3 次 → 暂时移除 60 秒
- 超时：connectTimeout=10s, timeout=120s
```

```php
class RpaRoutingDecider
{
    public function route(string $platformKey, string $action): string
    {
        // 自研 RPA — 稳定成熟渠道
        if ($this->hasMatureScript($platformKey)) {
            return 'native_rpa'; // baijiahao, toutiao, xiaohongshu, sohu, b2b168等15个
        }

        // Playwright MCP — 新渠道快速适配
        if ($this->isModernWebApp($platformKey)) {
            return 'playwright_mcp';
        }

        // Direct API — 有 Open API 的平台
        if ($this->hasOpenApi($platformKey)) {
            return 'direct_api'; // MediaBox API, 星链4SAPI等
        }

        // 默认兜底
        return 'native_rpa';
    }
}
```

**安全设计（多层防护）**：

```php
// Layer 1: PostgreSQL 用户级只读
// CREATE USER mcp_readonly WITH PASSWORD 'xxx';
// GRANT SELECT ON ALL TABLES IN SCHEMA public TO mcp_readonly;

// Layer 2: Laravel 全局 Scope（多租户过滤）
class McpTenantScope implements MCPToolMiddleware
{
    public function handle(ToolCallContext $context, Closure $next)
    {
        // 从 API Key 解析 workspace_id
        $workspaceId = $context->getWorkspaceId();

        // 注入全局 WHERE 条件
        Model::addGlobalScope('mcp_tenant', function (Builder $builder) use ($workspaceId) {
            $builder->where('workspace_id', $workspaceId);
        });

        return $next($context);
    }
}

// Layer 3: 禁止敏感表访问
const FORBIDDEN_TABLES = [
    'admins', 'admin_activity_logs', 'api_tokens',
    'system_logs', 'system_states', 'system_update_runs',
];

// Layer 4: 查询结果脱敏
class McpDataMasker
{
    public function mask(string $table, array $row): array
    {
        // 自动脱敏 api_key, credential_ciphertext, password 等字段
        foreach (['api_key', 'credential_ciphertext', 'password'] as $field) {
            if (isset($row[$field])) {
                $row[$field] = '***MASKED***';
            }
        }
        return $row;
    }
}
```
