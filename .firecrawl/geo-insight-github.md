## 🏢 关于 24K-GA

[Permalink: 🏢 关于 24K-GA](https://github.com/huanghfzhufeng/GEO-Insight#-%E5%85%B3%E4%BA%8E-24k-ga)

**24K-GA (Gold Agents)** 是一个专注于构建实用型 AI Agent、Agent 框架与真实世界自主系统的开源组织。

> **Open-source organization focused on building practical AI agents, agent frameworks, and real-world autonomous systems.**

我们致力于探索 LLM 在复杂场景下的落地实践，GEO Insight 是我们对于 AI 搜索引擎优化 (GEO - Generative Engine Optimization) 领域的探索性作品。

## 📖 项目介绍

[Permalink: 📖 项目介绍](https://github.com/huanghfzhufeng/GEO-Insight#-%E9%A1%B9%E7%9B%AE%E4%BB%8B%E7%BB%8D)

**GEO Insight** 是一个开源的品牌可见性分析平台，旨在帮助品牌主、市场人员和 SEO 专家量化品牌在国产大模型（如 Kimi、豆包、文心一言、DeepSeek）中的表现。

不同于传统的 SEO，GEO (Generative Engine Optimization) 关注的是品牌在 AI 生成内容中的出现频率、排名、情感色彩以及关联推荐。本项目采用 **"平行宇宙" (Parallel Universe)** 策略，通过 LangGraph 编排多智能体，模拟不同人设的用户在各个 AI 平台上的搜索行为，并利用 DeepSeek 作为 AI 裁判进行客观评分。

### 核心价值

[Permalink: 核心价值](https://github.com/huanghfzhufeng/GEO-Insight#%E6%A0%B8%E5%BF%83%E4%BB%B7%E5%80%BC)

- **量化 AI 印象**: 将模糊的 "AI 认知" 转化为具体的 GEO 评分 (0-100)。
- **竞品情报挖掘**: 利用 AI 自动发现潜在竞争对手，并进行多维度对比。
- **优化策略建议**: 针对低分项，自动生成可执行的内容优化建议。
- **跨平台仿真**: 同时模拟学术型、生活型、极客型等多种用户画像的搜索偏好。

## ✨ 功能特性

[Permalink: ✨ 功能特性](https://github.com/huanghfzhufeng/GEO-Insight#-%E5%8A%9F%E8%83%BD%E7%89%B9%E6%80%A7)

### 1\. 仪表盘与实时分析

[Permalink: 1. 仪表盘与实时分析](https://github.com/huanghfzhufeng/GEO-Insight#1-%E4%BB%AA%E8%A1%A8%E7%9B%98%E4%B8%8E%E5%AE%9E%E6%97%B6%E5%88%86%E6%9E%90)

直观展示各 AI 宇宙的评分概览。支持实时流式输出分析进度，从任务分发、模拟搜索到最终裁判评分的全过程可视化。

[![Dashboard Screenshot](https://github.com/huanghfzhufeng/GEO-Insight/raw/main/docs/images/dashboard.png)](https://github.com/huanghfzhufeng/GEO-Insight/blob/main/docs/images/dashboard.png)

### 2\. 真实竞品挖掘与对比

[Permalink: 2. 真实竞品挖掘与对比](https://github.com/huanghfzhufeng/GEO-Insight#2-%E7%9C%9F%E5%AE%9E%E7%AB%9E%E5%93%81%E6%8C%96%E6%8E%98%E4%B8%8E%E5%AF%B9%E6%AF%94)

集成 **DeepSeek** 深度推理能力，自动发现直接与间接竞品。支持：

- **AI 自动挖掘**: 根据品牌特性自动推荐 4-6 个强相关竞品。
- **多维对比雷达图**: 在知名度、好感度、相关性维度进行对比。
- **手动/自动双模式**: 即支持 AI 推荐，也支持手动指定竞品进行分析。

### 3\. 深度分析报告

[Permalink: 3. 深度分析报告](https://github.com/huanghfzhufeng/GEO-Insight#3-%E6%B7%B1%E5%BA%A6%E5%88%86%E6%9E%90%E6%8A%A5%E5%91%8A)

生成详细的单次分析报告，包含：

- **综合得分与排名**: 核心指标一目了然。
- **情感倾向分析**: 正面/负面/中性情感占比。
- **引用来源溯源**: 追踪 AI 回答背后的信息来源（知乎、公众号、36Kr 等）。
- **PDF 导出**: 支持生成高清 PDF 报告（含图表），便于分享与汇报。

[![Report Screenshot](https://github.com/huanghfzhufeng/GEO-Insight/raw/main/docs/images/report.png)](https://github.com/huanghfzhufeng/GEO-Insight/blob/main/docs/images/report.png)

### 4\. 企业级功能

[Permalink: 4. 企业级功能](https://github.com/huanghfzhufeng/GEO-Insight#4-%E4%BC%81%E4%B8%9A%E7%BA%A7%E5%8A%9F%E8%83%BD)

- **云端数据同步**: 跨设备、跨浏览器实时同步分析状态与偏好设置。
- **GEO 优化白皮书**: 内置行业白皮书，提供理论支持与优化方法论。
- **多模型支持**: 后端灵活集成 DeepSeek, Bocha, Jina 等最佳实践 API。

## 🚀 快速开始

[Permalink: 🚀 快速开始](https://github.com/huanghfzhufeng/GEO-Insight#-%E5%BF%AB%E9%80%9F%E5%BC%80%E5%A7%8B)

### 前置要求

[Permalink: 前置要求](https://github.com/huanghfzhufeng/GEO-Insight#%E5%89%8D%E7%BD%AE%E8%A6%81%E6%B1%82)

- Docker & Docker Compose
- API Keys:
  - [DeepSeek](https://im.deepseek.com/) (必需)
  - [Bocha](https://bochaai.com/) (必需，用于中文搜索)
  - Jina (可选)

### 安装步骤

[Permalink: 安装步骤](https://github.com/huanghfzhufeng/GEO-Insight#%E5%AE%89%E8%A3%85%E6%AD%A5%E9%AA%A4)

1. **克隆仓库**



```
git clone https://github.com/24K-GA/GEO-Insight.git
cd GEO-Insight
```

2. **配置环境变量**



```
cp .env.example .env
# 编辑 .env 文件，填入您的 API Keys
# DEEPSEEK_API_KEY=sk-...
# BOCHA_API_KEY=...
```

3. **使用 Docker 启动**



```
docker-compose up -d --build
```

4. **访问应用**
打开浏览器访问 [http://localhost](http://localhost/)
